# godoc

## Introducing

A tool for generating go package documents in Markdown format.

## Installation

```console
$ GO111MODULE=on go get github.com/iawia002/godoc
```

## Usage

```console
$ godoc github.com/iawia002/godoc/generator github.com/your/package/path
```
